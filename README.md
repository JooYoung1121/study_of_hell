# 지옥의 스터디

<br>

- Goal :
  1. Algorithmic Thinking
  2. 1day 1commit

<br>


- 방법 

1. 각 알고리즘 사이트에 대한 폴더 내에 본인 폴더를 만든다. 
2. 올려둔 문제를 최소 하루 한문제 이상 푼다. 
3. 풀고 각자 폴더에 업데이트 하고 문제가 생기거나 궁금점이 있다면 이슈로 올린다.
